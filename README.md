# 俄羅斯方塊 Tetris

純前端 HTML/CSS/JavaScript 打造的俄羅斯方塊,不需安裝、點開連結就能玩。

## 🎮 立即遊玩

**https://hao-github-0216.github.io/tetris/**

## 操作方式

| 動作 | 按鍵 |
| --- | --- |
| 左右移動 | ← → |
| 旋轉 | ↑ / X |
| 保留方塊 | C |
| 軟降 | ↓ |
| 硬降 | Space |
| 暫停 | P |

手機瀏覽時會自動切換成觸控按鈕。

## 功能

- 保留方塊(Hold Piece)
- 連擊(Combo)計分
- 音效
- 最高分紀錄

## 本機執行

不需要任何建置流程,直接用瀏覽器打開 `index.html` 即可,或啟動任一本機伺服器,例如:

```bash
python3 -m http.server
```

然後開啟 `http://localhost:8000`。
